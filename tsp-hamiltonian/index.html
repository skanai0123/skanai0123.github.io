How To Read This Page

Valid Tour Solver

This solver searches only over valid TSP tours. For small city counts it uses brute force, and for larger sets it switches to a heuristic route search. Because it compares only valid tours, changing penalty A usually does not change the best route.

QUBO Annealing

This solver minimizes the TSP QUBO directly over binary variables. It can temporarily visit invalid states, so changing penalty A can change the returned state and the recovered route.

Penalty A And Distance B

A controls how strongly the solver penalizes violating the one-city-per-slot and one-slot-per-city constraints. B scales the route distance term.

Why The Matrix Grows Fast

TSP uses variables x(i,t), so N cities create N^2 binary variables. The QUBO matrix therefore has size N^2 x N^2, which means the number of matrix entries grows like N^4.

Select cities on the map to keep the TSP instance small enough to inspect numerically.
