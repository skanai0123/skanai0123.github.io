QUBO / Ising Converter

Bidirectional converter between QUBO and Ising. The variable mapping is x = (1 - s) / 2 (s = 1 - 2x).

QUBO

\[ E_{\mathrm{QUBO}}(x) = x^\mathsf{T} Q x + c \]

Ising

\[ E_{\mathrm{Ising}}(\sigma) = \sum_{i\lt j} J_{ij}\sigma_i\sigma_j + \sum_i h_i\sigma_i + c \]

Variable Mapping

\[ x_i = \frac{1 - \sigma_i}{2}, \quad \sigma_i = 1 - 2x_i \]

Input JSON

QUBO: { "Q": [[...],[...]], "offset": 0 }

Ising: { "h": [...], "J": [[...],[...]], "offset": 0 }

Matrix tables are primary input. JSON is secondary.

Symbol Legend

  • x_i: QUBO binary variable (0 or 1)
  • \sigma_i (or s_i): Ising spin variable (-1 or +1)
  • Q: QUBO coefficient matrix
  • J: Ising coupling matrix between spins
  • h: Ising local field vector
  • c / offset: constant energy shift
  • n: number of variables / spins
